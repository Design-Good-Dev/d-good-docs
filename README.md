# 🎓 D-Good (졸업 전시 종합 아카이빙 서비스)

졸업 전시의 처음과 끝 모든 과정을 함께하는 **종합 아카이빙 서비스**입니다. 팀 프로젝트의 효율적인 기록과 관리를 위해 다양한 자동화 기능을 포함하고 있습니다.

## 🚀 주요 기능

- **종합 아카이빙**: 프로젝트 진행 과정의 모든 기록을 체계적으로 관리
- **자동 회의 기록 (Discord Recorder)**: 디코드 채널에서 음성을 실시간 녹음하여 파일로 저장
- **회의록 자동 변환**: 클로바 노트 요약 결과를 프로젝트 전용 마크다운 형식으로 변환

## 🛠️ 자동화 도구 사용법

### 1. 디스코드 녹음 봇
- `/automation` 폴더로 이동합니다.
- `.env` 파일에 `DISCORD_BOT_TOKEN`을 입력합니다.
- `python discord_recorder.py`를 실행합니다.
- 디스코드 명령어: `!join`, `!start`, `!stop`, `!leave`

### 2. 회의록 생성기
- 녹음된 파일을 클로바 노트에 업로드 후 요약 결과를 `.txt`로 저장합니다.
- `python process_minutes.py`를 실행하고 가이드를 따릅니다.

## 📂 폴더 구조
- `Meeting_Minutes/`: 회의록 보관함
- `Todo/`: 개인 및 팀 할 일 관리
- `automation/`: 자동화 스크립트 도구 모음
